# UI Layout Alignment Rules
- When implementing collapsible sections or layout changes, always inspect the styling and alignment of surrounding components (such as text positioning, margins, borders, and tabs).
- Ensure that labels, buttons, or titles do not shift or jump by even a single pixel horizontally or vertically when toggling states (such as opening or closing accordions).
- Prefer using identical, static absolute classes or maintaining structural layouts rather than switching between relative and absolute positions dynamically.
